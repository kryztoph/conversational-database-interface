Solution Architecture

CGI leverages proven enterprise technologies in a containerized, cloud-ready architecture designed for scalability, security, and reliability.

Technology Stack

PostgreSQL + pgvector

Industry-standard relational database with vector similarity search for RAG capabilities. Scales from small teams to enterprise deployments.

Large Language Models

OpenAI-compatible API supports cloud providers (OpenAI, Anthropic) or on-premises deployment (llama.cpp, Ollama) for air-gapped environments.

Python Application Layer

Modular architecture with separate concerns: database management, LLM client, and interactive CLI interface.

Docker Containerization

Fully containerized deployment ensures consistency across development, staging, and production environments.
